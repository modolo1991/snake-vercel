# Snake Deluxe upgrade — 2026-03-28

## What changed in this pass
- Split the old single-file game into a small static app structure.
- Added auth/account UI scaffold.
- Added browser-side Supabase client integration.
- Kept local save fallback so the game still works before Supabase is configured.
- Added profile sync model for:
  - best score
  - coins
  - owned skins/upgrades
  - equipped skin
  - active powers
- Added **Start fresh** reset flow.
- Added Supabase SQL schema and setup notes.

## Architecture choice
This stays a static Vercel app.
- no Node server required
- no secret backend required
- uses Supabase JS client in the browser
- uses Supabase anon key only
- row-level security protects each user's save row

## Main user-visible additions
- account section at the top
- sign in / sign up / magic-link actions
- explicit sync status
- cloud-sync aware save mode indicator
- reset-progress button that can wipe local and remote progress
